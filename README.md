# TechStore - Loja Virtual

Uma loja de tecnologia moderna e responsiva desenvolvida com HTML, CSS e JavaScript.

## 🚀 Como executar

1. Baixe os arquivos
2. Abra `index.html` no navegador
3. Pronto! A loja está funcionando

## 📱 Responsividade

- ✅ Mobile (até 480px)
- ✅ Tablet (481px - 768px)  
- ✅ Desktop (769px+)

## 🛠️ Tecnologias

- HTML5
- CSS3 (Grid, Flexbox)
- JavaScript
- Font Awesome Icons
